Navigation Structure

Search Architecture

How search functionality is organized and accessible throughout the application.

Search Types & Locations
Global Search
Header search box available on all pages. Searches members, journals, and galleries.
Members Search
/members/search - Advanced search with filters by name, relation, date, location.
Tree Search
Interactive search overlay in tree visualization. Highlights found nodes.
Content Search
Search within journals, gallery, and broadcasts with tags and categories.
Admin Search
Search users, access requests, audit logs with multiple filters.
Search Features & Capabilities
Arabic Text Support
Full support for Arabic search queries with proper RTL text handling.
Fuzzy Matching
Handles typos and variations in names and search terms.
Faceted Filters
Category, date range, status, and role-based filters on search results.
Search History
User's recent searches available in dropdown for quick re-access.
Saved Searches
Users can save frequent searches for quick access later.
Pagination
Search results paginated with configurable items per page (10, 25, 50, 100).
Search Query Syntax
name:"Ahmed Mohamed" - Exact name match
relation:father - Search by relationship
from:1980 to:1990 - Date range search
location:"Cairo" - Location search
verified:true - Filter by verification status
tag:"family event" - Search by tags
